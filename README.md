# Real Estate Property Listing Platform
This project is a *web-based real estate property listing platform* built using the *MERN stack (MongoDB, Express, React, Node.js)* with *Tailwind CSS* for styling and *Vite* for fast React setup.

The platform allows users to *browse, search, and filter properties,* while admins can *add, update, and manage property listings.* Itâ€™s designed to be *responsive, interactive, and scalable,* providing a smooth user experience for property buyers, sellers, and agents.

**Features:-**

* *User Authentication:* Register, login, and manage accounts securely

* *Property Listings:* View detailed property information including images, price, location, and description

* *Search & Filters:* Filter properties by type, price range, location, and other criteria

* *Admin Dashboard:* Add, edit, or delete property listings

* *Responsive Design:* Works seamlessly on desktop and mobile devices

* *Fast Performance:* Built with React + Vite for rapid load times and smooth interactions

**Technology Stack:-**

*1. Frontend:*

* React.js (with Vite for faster builds)

* Tailwind CSS for responsive styling

* React Router for navigation

* Axios for API calls

*2. Backend:*

* Node.js with Express.js

* MongoDB for database management

* JWT for authentication

**How It Works:-**

*1. Frontend (React + Vite + Tailwind CSS):*

* Displays property listings in a card/grid layout

* Provides search, filter, and sort functionality

* Handles user authentication and routing

*2. Backend (Node + Express + MongoDB):*

* Exposes REST APIs for fetching, adding, updating, and deleting properties

* Handles user authentication and data validation

* Stores property data in MongoDB

*3. Integration:*

* Frontend communicates with backend via Axios HTTP requests

* JWT tokens maintain secure sessions for logged-in users

* Admin dashboard interacts with the backend to manage listings

**How to Run:-**

*1. Backend Setup:*
````
cd backend
npm install
npm run dev
````

*2. Frontend Setup:*
````
cd frontend
npm install
npm run dev
````

*3. Open the provided Vite localhost URL in your browser to see the app.*
